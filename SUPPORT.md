# Support

## Questions and usage help

For general usage questions, start with:

- [README.md](README.md)
- [playground/README.md](playground/README.md)
- WordPress and Two Factor plugin documentation

## Bug reports

Use the GitHub bug report issue form and include:

- WordPress version
- PHP version
- Two Factor plugin version
- Activation mode: single-site, Network Activate, per-site multisite, or mu-loader
- Whether `FORCE_2FA_DISABLE`, role exclusions, or API allowlists are configured

## Security reports

Do **not** use public issues for vulnerabilities or authentication bypasses. See [SECURITY.md](SECURITY.md).
